eXist-db File Manager

Lightweight web component for browsing and managing eXist-db collections

Usage Instructions

  • Navigate: Click folders or use breadcrumbs
  • Select: Click items; Ctrl/Cmd for multiple
  • Copy/Cut: Ctrl+C/Ctrl+X or context menu
  • Paste: Ctrl+V or Paste button
  • Delete: Delete or context menu
  • Rename: F2 or context menu
  • Upload: Click Upload button
